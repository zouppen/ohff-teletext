# Samples

Rendered HTML previews for common page states:

- `01-full-mixed.html`: recent spots, agendas, and previous activations today
- `02-no-data.html`: no OHFF spots or agendas
- `03-agendas-only.html`: agendas without spots
- `04-previous-activations-today.html`: older same-day spots only
- `05-recent-spots-no-agendas.html`: recent spots without agendas

Each HTML file embeds the Teletext50 font so it can be opened directly in a
browser without installing fonts.
